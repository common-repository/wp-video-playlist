# WP Video Playlist

List of features of plugin.

* You can add Video item in playlist from Media files.
* You can add Video item in playlist from Youtube.

== Installation ==
Download the plugin .zip file

Login in to admin Click Plugins -> Add New -> Upload

Find WooCommerce Featured Reviews Wordpress Plugin in plugin list and activate it.

Goto "<strong>Video Player</strong>", add video and enjoy.

== Frequently Asked Questions ==

= How to add it =
1. Open "<strong>Video Player</strong>" from the sidebar.
2. After that click on "<strong>Add Video</strong>" beside Videos.
3. Then click on "<strong>Add Video</strong>" button and then select the video you want to add.
4. You also can add a custom title and URL. The first field for the title and the second one for the video URL.
5. After all this click on save and grab the shortcode and paste it where you want to show player.
